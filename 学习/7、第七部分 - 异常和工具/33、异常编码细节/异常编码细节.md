## try/except/else 语句

注：在 Python 2.5 以后，`except` 和 `finally` 可以混在同一个 `try` 语句中。

```Python
try:
    <statements>
except <name1>:
    <statements>
except <name2, name3>:
    <statements>
except <name4> as <data>:
    <statements>
except:
    <statements>
else:
    <statements>
```
